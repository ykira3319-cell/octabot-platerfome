
# OCTABOT Platform

[![Vercel](https://vercelbadge.vercel.app/api/octabot-platform)](https://vercel.com/octabot-platform)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![Node.js](https://img.shields.io/badge/Node.js-18.x-brightgreen)](https://nodejs.org/)

Plateforme interactive pour créer vos propres commandes JavaScript avec paiement et chatbot en temps réel.

---

## 🚀 Fonctionnalités

- Page de **chargement** avant l’affichage du site.
- Page de **bienvenue**.
- **Inscription utilisateur** simple (nom d’utilisateur et photo optionnelle).
- **Paiement via WhatsApp** pour activer le compte.
- **Notification en temps réel** pour confirmer le paiement.
- **Chatbot CMD JS** où l’utilisateur peut tester ses propres commandes JavaScript.
- **Page Admin** pour confirmer ou refuser les paiements.
- **Mise à jour instantanée** pour notifier les utilisateurs des nouvelles fonctionnalités.
